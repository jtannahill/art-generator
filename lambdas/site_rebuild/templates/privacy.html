Privacy Policy

Last updated: March 16, 2026

Overview

art.jamestannahill.com is a generative art project by James Tannahill. This site is designed to be privacy-friendly. We do not collect personal information, require accounts, or use tracking cookies.

What We Don't Collect

  • No personal information (name, email, phone)
  • No user accounts or login credentials
  • No advertising pixels or retargeting
  • No form submissions or user-generated content

Analytics

This site uses Google Analytics (GA4) to understand how visitors interact with the gallery — which artworks are viewed, how people navigate the archive, and general traffic patterns. Google Analytics collects anonymized usage data including page views, approximate location (country/city level), device type, and referral source. This data is used solely to improve the site experience. No personally identifiable information is collected. You can opt out of Google Analytics by installing the Google Analytics Opt-out Browser Add-on.

Infrastructure

This site is served via Amazon CloudFront (CDN). CloudFront may set technical cookies for performance and security purposes (such as session affinity). These are not used for tracking or advertising. Standard web server logs (IP address, user agent, requested URL) may be retained by AWS infrastructure for security and operational purposes.

Third-Party Services

The "Generate New Art" feature makes requests to AWS Lambda endpoints. No personal data is transmitted — only the selected artist preference. Weather data is sourced from Open-Meteo (public API). Satellite imagery is sourced from Copernicus/ESA (public data).

Artwork Downloads

SVG artwork files are available for download. Downloaded files contain no tracking mechanisms, metadata, or personal information. Artwork is licensed under CC BY-NC-ND 4.0.

Contact

For privacy-related inquiries: art@jamestannahill.com
